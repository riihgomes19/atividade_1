# 🚀 Atividade 1

Este repositório contém o desenvolvimento da **Atividade 1**, um projeto inicial voltado para a prática de conceitos básicos de desenvolvimento web utilizando HTML, CSS e publicação com GitHub Pages.

## 🌐 Acesse o projeto

👉 https://riihgomes19.github.io/atividade_1/

## 📌 Sobre o projeto

O objetivo desta atividade é aplicar na prática os fundamentos da construção de páginas web, incluindo:

- Estruturação de conteúdo com HTML
- Estilização com CSS
- Organização de arquivos
- Publicação utilizando GitHub Pages

Este projeto representa um dos primeiros passos no aprendizado de desenvolvimento front-end.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Git e GitHub
- GitHub Pages

## 📂 Estrutura do projeto

atividade_1/

├── index.html

└── README.md

## 🎯 Objetivos de aprendizagem

- Compreender a estrutura básica de uma página web
- Criar layouts simples
- Trabalhar com links, imagens e textos
- Publicar um projeto na web
